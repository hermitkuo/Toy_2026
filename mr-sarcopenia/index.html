01

Background

Why a Mendelian randomization study?

臨床觀察

Sarcopenia 在慢性腎臟病 (CKD) 病人盛行率 30–60%， 與死亡率、透析依賴、住院率上升有關。

觀察研究無法分辨：

  • 腎功能本身的影響
  • CKD 常用藥物的混淆 (confounding by indication)
Research Question

腎功能本身是否因果性影響骨骼肌？常用腎科用藥 （statin / SGLT2i / ACEi / ARB / MRA / loop）是否 獨立於腎功能影響肌肉？

研究取徑
  • Bidirectional MR → 雙向因果
  • Drug-target MR → cis-variant 模擬藥物終生暴露
  • MVMR → 中介路徑 (SBP / LDL / HbA1c)

Instrument selection

GWAS-wide + drug-target cis-variant strategies

Genome-wide instruments
  • 顯著性閾值：p < 5 × 10⁻⁸
  • LD clump：r² < 0.001 / 10,000 kb
  • Reference panel：1000 Genomes EUR
  • Instrument F-statistic > 10（所有暴露）
Drug-target cis-variants
  • HMGCR / SLC5A2：±100 kb · relaxed r² < 0.3 / 500 kb
  • ACE / AGTR1 / NR3C2 / SLC12A1：採文獻 lead SNP
  • rs4291 · rs4343 · rs5186 · rs5522 · rs2074299 · rs7178344
  • Effect from Evangelou 2018 SBP GWAS
Key Design

eGFR-cystatin C
as a parallel instrument

問題：eGFR-creatinine 方程式內含血清 creatinine， 而 creatinine 由骨骼肌產生 → instrument 受肌肉量混淆。

解法：Cystatin C 由所有有核細胞穩定分泌，與肌肉量無關 → 作為平行 instrument。

推論策略：若 cysC 與 crea 一致 → 真實腎功能因果； 若分歧 → crea 訊號為 instrument bias。

Drug-target MR

六類腎科用藥之 cis-variant 結果

Null (safe) Borderline (needs replication) — — — OR = 1 reference
無不利證據

Statin (HMGCR) · SGLT2 inhibitor (SLC5A2) · Loop diuretic (SLC12A1) 對肌肉量與肌力均無因果不利證據，臨床可安心使用。

Borderline 訊號（需 replication）

Take-home messages

本研究的四個主要訊息

01
正向發現

腎功能下降會因果降低四肢瘦肉量 (ALM, OR 0.73, p=0.008)。 以 cysC instrument 排除肌肉源 creatinine 干擾後仍穩健。

02
方法學貢獻

eGFR-crea / eGFR-cysC 雙 instrument 對照成功揭露： 「握力 ↔ 腎功能」雙向關聯其實為 instrument bias 假象。

03
藥物安全訊號

常用腎科用藥（statin · SGLT2i · loop diuretic）對肌肉 無因果不利證據； ACE 與 NR3C2 borderline 訊號待 replication。

04
執行進度

結案報告 67% STROBE-MR；manuscript v1 已成稿（Abstract + IMRaD）； 剩 MVMR B2 與 Discussion 4.3–4.6 補完。
