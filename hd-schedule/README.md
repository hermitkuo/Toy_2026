# hd-schedule — 花蓮慈院血透室班表頁

一套以 **純靜態 HTML + Google Sheet** 為後端的血液透析（HD）排班工具集，部署於 GitHub Pages。
醫師用來查班表 / 找空檔 / 看個人時段 / 看統計；排班者用來自動排班、人工微調、一鍵發布。

資料流：前端以 **gviz JSONP** 直接讀公開 Google Sheet（無自架伺服器）。排班者的發布動作透過
一支 Apps Script Web App（`publish_webapp.gs`）寫回試算表分頁。

## 頁面一覽

| 檔案 | 對象 | 說明 |
|------|------|------|
| `index.html` | 醫師 / 行政 | 主查詢頁，含四個分頁（見下） |
| `schedule.html` | 排班者 | 排班「編輯頁」：自動排班 → 人工微調 → 確認鎖定 → 一鍵發布 |
| `leave.html` | 醫師 | 填請假（整天）／有事（單一時段）— 目前 mock，尚未連後端送出 |
| `preference.html` | 醫師 | 填 S6 偏好（星期幾 × 院內時段早/中/晚 + 優先序）— 目前 mock |
| `publish_webapp.gs` | Apps Script | `doPost`（token 驗證）：把排班草稿寫入「自動排班草稿」分頁 |
| `setup_settings_sheet.gs` | Apps Script | 一鍵建立四個設定分頁（含醫師配額）+ 資料驗證 |
| `排班規則_草稿.md` | 文件 | 排班臨床規則 v1.0：硬限制 H1–H9／軟限制 S1–S7 |
| `CLAUDE.md` | 文件 | 產品/UI 設計取向（列印可讀性、個人行事曆只呈現事實等） |

### `index.html` 的四個分頁

| 分頁 | 功能 |
|------|------|
| 📅 **找班表** | 依日期列出當月班表。院區篩選器（花蓮／玉里／關山／聯安，可多選、欄列雙向收斂）、醫師高亮、🖨 整月列印（A4 直式、醫師縮寫 14px、彩色但黑白友善） |
| 🔎 **找空檔** | 讀輪值母表正規化，自動算「有空／全天有空」；可標「有事」時段 |
| 🗓️ **找我有空** | 個人週行事曆，只呈現事實、不標偏好優先序 |
| 📊 **找統計** | 跨年統計（依 2026-01-01 分界合併歷史表與現行表）：院內 HD 月班數、院區支援月班數、大抽週六次數、國定假日值班清單 |

## 資料來源（Google Sheet）

前端讀取的公開試算表分頁：

| 用途 | gid |
|------|-----|
| 現行班表（≥ 2026-01-01） | `1837573287` |
| 歷史表（< 2026-01-01） | `834332171` |
| 國定假日表 | `600263934`（欄序：日期／週幾／節日名稱／醫院是否放假／**HD醫師非週末出勤**） |
| 自動排班草稿（閱覽頁 B 模式，預設關閉） | 填入 `index.html` 的 `DRAFT_GID` 即啟用 |

> 國定假日計入條件用「HD醫師非週末出勤」旗標：1／是 → 計；0／空白 → 淡化不計。

## 排班規則（摘要，詳見 `排班規則_草稿.md`）

- **H5 值班輪值**：徐王賴林郭鄧翔，一人一天、週末連兩天、跨月延續，值班日可兼外派。
- **H6** 玉里／關山／聯安 同名展開；**H7** 聯安掛名（祺／翔／超／禮）。
- **H8/H9** 掛名檢核：花蓮HD／特約 不得與 聯安掛名／玉里／關山／請假 同人，衝突拆早中晚。
- **S1–S7** 軟限制：依母表/輪值、避免連續（週六例外）、S6 偏好等。

## 本機開發

純靜態頁，無建置步驟。直接以瀏覽器開啟 `index.html`，或起一個本機伺服器：

```sh
python3 -m http.server 8000   # 然後開 http://localhost:8000/index.html
```

驗證用 Playwright（本專案慣例）跑四頁與換月/篩選/列印路徑。
